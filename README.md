# M_马赛克赛马_M · 个人主页

基于 frontend-design 设计指南打造的个人网站。

## 设计说明

- **风格**：现代、简洁、浅色系
- **字体**：Noto Serif SC（标题）+ DM Sans（正文）
- **配色**：奶油白背景 + 青蓝点缀 + 沙金色辅助（呼应 Been 品牌「光与尘」）
- **布局**：留白充足、单页滚动、分区清晰

## 文件结构

```
WebPage/
├── index.html       # 个人主页
├── been-qrcode.png  # Been 小程序二维码
├── 项目计划书.md    # Been 项目参考文档
└── README.md
```

## 部署

可将 `index.html` 与 `been-qrcode.png` 部署至：
- **GitHub Pages**：参考 `../personal-site/DEPLOY.md`
- **Gitee Pages**：同理，上传后开启 Pages 服务
